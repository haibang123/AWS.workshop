## Create Amazon SNS

1. Search for and select the **SNS** service.

   ![SNS](/images/1/3.8.png?width=90pc)

2. Select **standard**.
3. Select new sns

   ![SNS](/images/1/3.9.png?width=90pc)

3. Click **Create topic**.

   ![SNS](/images/1/3.10.png?width=90pc)
